Comparison of Some Programming Languages

We will compare some of my favorite programming languages:
  • Scala
  • C
  • Python
Language Type System Standard Runtime Memory Management
Scala Statically-typed, with type inference JVM Automatic memory management provided by JVM
C Statically-typed Compiled to native code Memory must be managed by programmer
Python Dynamically-typed Interpreter (JIT implementations do exist) Automatic memory management provided by interpreter
